Project Vision

AI-Based Dropout Prediction and Counselling Platform

This system blends risk prediction, explainable AI, and intervention workflows to help educators detect risk early and respond with targeted support.

Core Features
AI Predictions

RandomForest-based dropout risk scoring from multi-factor student signals.

Explainable AI

SHAP feature attribution for transparent risk reasoning per student.

Modular Architecture

Blueprint-driven Flask layers for routes, controllers, services, and repositories.

Operational Dashboard

Role-oriented dashboards for monitoring trends, alerts, and interventions.

Counselling Recommendations

Rule-guided intervention recommendations based on active risk factors.

Student Assistant

Built-in chatbot support for students seeking immediate guidance.

System Goals

Early Detection

Actionable Support

Transparent Decisions

Data Workflow

1. Ingest student and activity data

2. Score risk using trained models

3. Explain top contributors

4. Trigger alerts and interventions

Technology Stack

Backend

Flask, SQLAlchemy

Frontend

Jinja2, Bootstrap 5, Chart.js

ML + XAI

Scikit-learn, SHAP, Pandas, NumPy
