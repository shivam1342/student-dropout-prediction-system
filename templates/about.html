About the AI-Based Dropout Prediction System

This project is a comprehensive Flask application designed to identify students at risk of dropping out and provide actionable insights for counselling and intervention.

Core Features
  • AI-Powered Predictions: Utilizes a machine learning model (RandomForest) to predict dropout risk based on various student metrics.
  • Explainable AI (XAI): Integrates SHAP (SHapley Additive exPlanations) to identify the top factors contributing to each student's risk score, making the AI's decisions transparent.
  • Modular Architecture: Built with Flask Blueprints for a clean and scalable separation of concerns (routes, controllers, models).
  • Interactive Dashboard: A central dashboard provides a high-level overview of student risk trends, summary statistics, and quick access to high-risk profiles.
  • Counselling Portal: Automatically generates rule-based counselling recommendations based on a student's specific risk factors.
  • Student Management: Full CRUD (Create, Read, Update, Delete) functionality for managing student records.
  • Virtual Assistant: A simple, rule-based chatbot to provide initial student support and guidance.
Technology Stack
  • Backend: Flask, SQLAlchemy
  • Frontend: Jinja2, Bootstrap 5, Chart.js
  • Machine Learning: Scikit-learn, Pandas, NumPy
  • Explainability: SHAP
  • Database: SQLite
